# Paula González — Designer Portfolio

A fashion editorial portfolio website for Paula González, deployed on Azure Static Web Apps with images hosted on Azure Blob Storage.

## Live Site
Deployed automatically via GitHub Actions on push to `main`.

## Architecture
- **Hosting**: Azure Static Web Apps (Free tier)
- **Images**: Azure Blob Storage (`stpaulaportfolio` / `images` container)
- **CI/CD**: GitHub Actions (auto-generated by Azure)
- **Resource Group**: `rg-paula-portfolio` (eastus2)

## Managing Images

### Upload new images
```bash
az storage blob upload \
  --account-name stpaulaportfolio \
  --container-name images \
  --file ./my-photo.jpg \
  --name my-photo.jpg
```

### List images
```bash
az storage blob list --account-name stpaulaportfolio --container-name images --output table
```

### Or use Azure Storage Explorer
Download from: https://azure.microsoft.com/products/storage/storage-explorer/
